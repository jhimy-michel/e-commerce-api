# JM Online shop (e-commerce-api)

This project was created using Loopback.

[![LoopBack](https://github.com/strongloop/loopback-next/raw/master/docs/site/imgs/branding/Powered-by-LoopBack-Badge-(blue)-@2x.png)](http://loopback.io/)

API created for the JM Online shop application (the JM stands for Jhimy Michel, that is my name btw).

This is the stack I used to build this API:

1. TypeScript
2. Jest
3. Eslint
4. Prettier
5. Husky
6. MongoDB
7. Stripe
